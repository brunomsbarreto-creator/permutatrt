# Permuta Fácil — Como publicar no Vercel

## Passo a passo

1. Acesse https://github.com e crie uma conta (se não tiver)
2. Crie um repositório novo chamado `permutafacil`
3. Faça upload de todos os arquivos desta pasta
4. Acesse https://vercel.com e entre com sua conta GitHub
5. Clique em "Add New Project"
6. Selecione o repositório `permutafacil`
7. Clique em "Deploy"
8. Pronto! Você terá um link como https://permutafacil.vercel.app

## Estrutura dos arquivos
- `src/App.js` — código principal do app
- `src/index.js` — entrada do React
- `public/index.html` — HTML base
- `package.json` — dependências
- `vercel.json` — configuração do Vercel
